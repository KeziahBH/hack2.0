Online Learning Platform — Hack2.0

A simple and interactive online learning platform built using HTML, CSS, and JavaScript.
It includes user authentication, course listings, video pages, quizzes, and dynamic content loading.

 Features
 1. User Authentication System

User Register

User Login

Password validation

Stores user data in localStorage

Protects pages from unauthorized access

Logout functionality

 2. Dynamic Course Pages

Courses loaded from a courses.json file

Course details display dynamically

“Start Learning” action

Clean UI using cards and images

 3. Learning Content

HTML course pages (k1.html, k2.html…)

Java course pages

Video classes page

Theory + examples + demo code for each course
